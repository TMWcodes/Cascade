### CSS basics

```
selector {
property1: value1;
property2: value2;
}

selector priority (lowest to highest) - element, class, id.
inline overwrites style sheet
css selectors

strong {
color: blue;
}

class
.class-name {
property: value;
}

id
#id {
property: value;
}

combinations
.selector-1.selector-2 {

}

ancestor
.ancestor .child {

}

combine
.big, .large {

}

everything

- {
  font-family: Arial;
  }
```
